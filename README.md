# CALENDAR APP
# TECNOLOGIAS React + TypeScript + Vite

Aplicacion web sobre creacion de eventos en el calendario, con personalización de la descripción del evento. Uso de tecnologias(MERN), TypeScripts, vite, se aplican conceptos (Redux, Store, Slice, Manejo de fechas, hooks personalizados para logica empresarial)

## DEVELOPMENT INSTRUCCIONES: 

1. Renombrar el archivo .env.template a .env
2. Hacer los cambios respectivos en las variables de entorno.

```
VITE_API_URL=http://localhost:4000/api
```
